**PixelCrush** is a Chrome extension for Gmail created by Michaël Berger. It detects and blocks common email trackers (pixels, known tracking URLs, and related patterns) and shows thread-level badges in your inbox. It is free and open source.

## Development

- Load unpacked from the repository root (folder containing `manifest.json`) at `chrome://extensions`.
- Build a Chrome Web Store zip: `.\scripts\package-for-store.ps1` → `pixelcrush-store.zip`.
