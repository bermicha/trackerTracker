PixelCrush — Windows installer options
======================================

FOLDER INSTALL (no .exe)
------------------------
  Installs to:
    %LOCALAPPDATA%\PixelCrush\extension

  From repo root (PowerShell):
    powershell -ExecutionPolicy Bypass -File .\installer\Install-PixelCrush.ps1

  Or: double-click Install.bat in this folder.

  Uninstall:
    powershell -ExecutionPolicy Bypass -File installer\Uninstall-PixelCrush.ps1

INNO SETUP (.exe installer)
----------------------------
1. Install Inno Setup 6: https://jrsoftware.org/isinfo.php
2. Open installer\PixelCrush.iss in Inno Setup Compiler.
3. Build → Compile. Output: installer\PixelCrush-Setup.exe
4. Run PixelCrush-Setup.exe on any PC with Chrome installed.

Shortcuts launch Google Chrome with:
  --load-extension="<install>\extension"

So the extension loads for that Chrome window without using "Load unpacked" manually.
(Chrome Web Store install is still the standard for auto-updates; this is for sideloading.)
